### In the project directory, you can run:

- Fork the project and clone it locally.
- In the project directory, ou can follow the steps below to download the dependencies:

  Install with npm:

  ```sh
  npm i
  ```

  Install with yarn:

  ```sh
  yarn
  ```

- In the project directory, you can run:

  Run with npm:

  ```sh
  npm run
  ```

  Run with yarn:

  ```sh
  yarn start
  ```
