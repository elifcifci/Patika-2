## Testing Exercises
- Created tests for components.

### In the project directory, you can run:
- Project dependencies download for backend and client:
    ```javascript
        cd backend
        npm i

        cd ..
        cd client
        npm i
    ```

- Redis run:
    ```
        redis-server
    ```

- Backend run:
    ```
        cd backend
        npm run dev
    ```

- Client run:
    ```
        cd client
        npm start
    ```
